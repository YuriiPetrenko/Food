# Food

Runs the app in the development mode.\
Open any free local server such as XAMPP / MAMP, drop project in folder **htdocs** \
To run json-server use: `npx json-server db.json`

## Techtology:

- **HTML** 
- **CSS** 
- **JavaScript** 
- **ES6**
- **Promises**
- **JSON-server**
- **AJAX**
- **Fetch API**
- **Webpack(Modules)**
- **NPM**
- **localstorage**
